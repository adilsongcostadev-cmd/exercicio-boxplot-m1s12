Coleção de exercícios resolvidos de **Estatística Aplicada à Análise Preditiva**, Semana 12 do curso **Desenvolvimento de IA para Análise Preditiva** (Programa SCTEC / SENAI-SC — Carreira Tech, Etapa Profissionalizar).

## 📂 Exercícios

| # | Exercício | Conteúdo | Pasta |
|---|---|---|---|
| 01 | Box Plot | Cinco números resumidos, AIQ, limites de discrepância, outliers e construção do box plot | [`01-boxplot/`](./01-boxplot/) |
| 02 | Média, Moda e Mediana | Dados agrupados em classes: ponto médio, porcentagem acumulada e medidas de posição central | [`02-media-moda-mediana/`](./02-media-moda-mediana/) |

Cada pasta contém o enunciado resumido, a resolução passo a passo em Python e o respectivo README com os resultados e a interpretação.

## 🚀 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/adilsongcostadev-cmd/exercicios-estatistica-m1s12.git
```

2. Instale a dependência (necessária apenas para o exercício 01):

```bash
pip install matplotlib
```

3. Entre na pasta do exercício desejado e execute o script:

```bash
cd 01-boxplot
python exercicio_boxplot.py
```
